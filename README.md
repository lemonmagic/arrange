# arrangement
## 项目结构
- assets 静态文件
- components 
  - arrangeWay 服务编排方式选择组件
  - guide 引导页组件
  - home 首页相关组件
  - increase 增加产品相关组件
  - setting 设置组件
  - show 展示已编排产品组件
- plugins
  - axios 网络请求封装
  - driver 引导库
  - elementui elementui组件库按需引入
  - websocket 长连接封装
- router 路由
- store vuex公共组件
- views 
  - home 服务编排端系统页
  - login 登录页

## Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
